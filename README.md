## Available Scripts

In the project directory, you can run:

### Development

First you are need build core:
```
$ yarn core:build
```

Start react development server:
```
$ yarn react:serve
```

Start vue development server:
```
$ yarn vue:serve
```

### Testing

Run unit tests:

```
$ yarn test
```
